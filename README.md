# Budget_app
🚀 Project Complete: Built a Clean-Code Budget Tracker in Python! 🐍

I just finished building an automated Budget App that tracks category spending, handles transfers between accounts safely, and generates a dynamic terminal-based data visualization chart. 📊

This project was a great exercise in core software engineering principles:
✅ Object-Oriented Programming (OOP): Designing robust class blueprints with encapsulated data.
✅ Input Validation & Error Handling: Building safety checks (`check_funds`) into every transaction method to ensure data integrity.
✅ Dynamic String Formatting: Manipulating matrices, vertical text padding, and right/left alignments without relying on external UI packages.

Instead of just jumping straight to a solution, I took the time to break down the algorithm step-by-step—understanding how Python maps memory under the hood, how nested loops build grid matrices, and how floor division optimizes data math. 

💡 Continuous growth is my main priority, and seeing this pass 100% of its automated test suite feels incredibly rewarding!
